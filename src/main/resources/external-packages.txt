# 外部ツール連携とみなすパッケージ接頭辞の拒否リスト。
# ここに列挙したパッケージを import できるのは repository/*.java のみ。
# それ以外のレイヤー（top / layer* / dto / log / util / validation / constants / Application）で
# import するとエラーになる。
#
# 書式:
#   - 1行に1つのパッケージ（またはクラス）接頭辞を記載
#   - 「接頭辞」と完全一致、または「接頭辞.」で始まる import にマッチ
#   - '#' 以降は行コメント。空行は無視
#
# 差し替え方法（再ビルド不要）:
#   - 実行時のカレントディレクトリに external-packages.txt を置く、または
#   - java -Dexternal.packages=/path/to/list.txt -jar app.jar <project>
